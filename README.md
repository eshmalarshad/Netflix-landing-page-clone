# Netflix-landing-page-clone

This project is a front-end clone of the Netflix landing page built using HTML and CSS. It replicates the design, layout, and responsiveness of the official Netflix homepage.

## Features
- Responsive design for different screen sizes
- Modern UI similar to Netflix
- Hero section with call-to-action
- FAQ section with interactive styling
- Clean layout with images and video elements

## Technologies Used
- HTML5
- CSS3 
- Google Fonts

##  Preview
This project mimics the look and feel of Netflix's homepage including sections like:
- Hero banner
- Feature sections
- FAQ section
- Footer

##  Purpose
This project was built for learning and practicing front-end web development skills, especially layout design and responsiveness.
